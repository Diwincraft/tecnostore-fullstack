# TecnoStore Fullstack

Sistema web fullstack desarrollado con Flask, SQLite, Bootstrap y JavaScript.

El proyecto simula una tienda tecnológica moderna con catálogo dinámico, carrito de compras, panel administrativo y autenticación de usuarios.

---

# Características

- Catálogo dinámico de productos
- Carrito de compras
- Panel administrativo
- Login de administrador
- CRUD de productos
- Base de datos SQLite
- Interfaz responsive con Bootstrap
- Gestión de sesiones
- Integración frontend y backend

# Instalación

## Clonar repositorio

```bash
git clone https://github.com/TUUSUARIO/tecnostore-fullstack.git
```

---

## Entrar al proyecto

```bash
cd tecnostore-fullstack
```

---

## Instalar Flask

```bash
pip install flask
```

---

## Ejecutar aplicación

```bash
python app.py
```
